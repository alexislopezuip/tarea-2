# Tarea 2

Repositorio creado para practicar el uso de GitHub, Git y el control de versiones mediante ramas.

## Descripcion

Este proyecto contiene un archivo de Python llamado `script.py` con una funcion que imprime un saludo y otra funcion que muestra la suma de `1 + 1`.

## Archivos

- `script.py`: script principal de la tarea.

## Ramas utilizadas

- `main`: rama principal del repositorio.
- `modificaciones`: rama creada para agregar la funcion de suma.

## Como ejecutar

```bash
python script.py
```

## Resultado esperado

```text
Hola, GitHub
2
```
